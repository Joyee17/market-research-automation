# 内部数据输入模板
## 市场调研自动化工具 · 必填信息包
## 版本：v1.0 | 适用：六种调研类型（A/B/C/D/E/F）

> **使用说明**
> 1. 每次开始调研前，填写「必填区域」（Section 1-3）
> 2. 根据调研类型，填写对应的「类型专属区域」（Section 4）
> 3. 填完后将整个文档内容粘贴给 Claude，工具会自动完成其余工作
> 4. 不知道的字段填「未知」，工具会标注该信息缺口并建议获取方式

## ═══ 填写示例（CPV产品参考）═══

```
【决策问题】
确定砷化镓CPV组件在美国/澳大利亚/智利/南非中优先进入的市场，
以及第一阶段的目标客户类型和商业模式

【产品/服务名称】
砷化镓CPV组件（GaAs Concentrator Photovoltaics）

【核心技术指标】
指标1：组件效率 37%（PM-1B型号），行业晶硅均值22-25%
指标2：温度系数 -0.15%/℃，晶硅为 -0.25%~-0.30%/℃
指标3：聚光倍数 865X（PM-1B），使用寿命30年
指标4：单位面积发电量比晶硅高约72%（高DNI区域实测）

【产品发挥优势的必要条件】
DNI > 6.5 kWh/m²/天、环境温度 > 35°C、高直射光比例、
需要高精度双轴追光系统配合

【产品明确不适用的场景】
DNI < 5.0 的温带地区（如欧洲北部、中国东北）；
屋顶分布式（需要平铺，追光系统无法安装）；
多云/散射光为主的气候区

【公司资源约束】
无海外团队；无当地合作伙伴；产品有PM-1A/PM-1B/AM-1三款型号；
尚无海外客户订单；原目标中东市场，因局势变化寻找替代市场
```
